# Task Management Dashboard

A simple Task Management app built using React.

## Features
- Add task
- Mark task as completed
- Delete task
- Filter tasks (All / Completed / Pending)
- Data stored in localStorage

## Tech Stack
- React (Hooks)
- JavaScript
- CSS

## Run Locally

```bash
git clone https://github.com/imarani494/dicuse.git
cd dicuse
npm install
npm run dev
